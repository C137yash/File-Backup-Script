# 🗃️ Backup Script

Automatically copies a folder, zips it with a timestamp, and logs the process.

## 🚀 Features
- ✅ Copies source folder to backup folder
- ✅ Zips backup with timestamp
- ✅ Logs success/failure to `backup_log.txt`
- ✅ Deletes unzipped folder to save space

## 📥 How to Run
1. User is required to enter - `SOURCE_FOLDER` and `BACKUP_FOLDER` in `backup_script.py`
2. Run:
   ```bash
   python backup_script.py
